The Main Menu #

This section will now show you how the app works in practice. Please follow the Installation Guide in Chapter 2.0 before continuing.

  1. Launch Latex Curriculum Vitae from your desktop.

  2. The user interface opens, displaying:

You have two options. To start writing your application, you can click on the button and follow the documentation below ???. If you want to see or change your user settings, select and read more on ???. opens the Database Window.

Write Your Application #

Use this windwo to write your job application. To begin:

  1. In the "Job Title" box, place the title of your new job.

  2. The company name is already known, then type it in the "Company Name" box.

  3. You can include more information about the company in the address boxes. If you do not know the street, zip code, and city, you can leave it blank.

  4. Now you can enter the information for the contact person. If you do not have a name, then simply leave this box blank. In this situation the app will write "To whom it may concern" as a salutation (this salutation will also be used if you select a gender of "Other/Neutral"). In all other cases the app produces a "Dear Mr. Bean" or "Dear Mrs. Bean" salutation.

    The "Application details" range are pure optional. In the "URL" field you can add the job offer URL. It will be placed in the CSV file for future reference.

    If you set the checkbox "Initiative application", the Subject from the letter_of_applicaton.tex will be changed from "Application as" to "Initiative application as".

  5. There are two options for using the app. In the case where you enter a email address and click , the app will compile all files and email it directly (if you have set the SMTP settings in the Installation Guide. See ???) with you in BCC. If you leave the email box blank and click Generate, the app will also compile all files, but in this case will open Windows Explorer showing the compile folder (AppData/Local/Temp/latex_curriculum_vitae) directory. Every file that you can see in this directory will be deleted with each new iteration; however, you now have the option to upload the files yourself to a job application website.

  6. Now your application is done! You can now proceed to writing another.

  7. By clicking on you will exit the app. If you are select the button: you will be forwarded to this online documentation.
